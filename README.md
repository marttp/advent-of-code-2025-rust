# Advent Of Code 2025 in Rust

By Thanaphoom Babparn (Mart)

This repository used for learning in Rust by attending [Advent of Code 2025](https://adventofcode.com/2025)

## Command

### Create new day
```shell
# Example: cargo new day01
cargo new dayxx
```

### Run 
```shell
cd dayxx
cargo run --bin part1
cargo run --bin part2
```
